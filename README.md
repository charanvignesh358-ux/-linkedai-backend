# LinkedAI Backend

This space runs the LinkedAI backend, which orchestrates browser automation (Playwright) and Firebase Admin tasks.

## Setup Instructions

Make sure this Space is set to **Private** to protect your `serviceAccountKey.json`.
You will need to configure your secrets or upload your `serviceAccountKey.json` directly to the `firebase/` directory inside your Space files.
